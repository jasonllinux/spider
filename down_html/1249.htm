科技部基础研究司关于召开2012年全国地方基础研究工作会的预通知

科技部门户网站  www.most.gov.cn      2012年10月08日    

国科基函【2012】28号

各省、自治区、直辖市、计划单列市科技厅（委、局），新疆生产建设兵团科技局：
    为贯彻落实全国科技创新大会精神，促进地方基础研究管理改革，加强区域创新能力建设，经研究，定于2012年10月25-26日在广西桂林召开2012年全国地方基础研究工作会议。现将有关事宜预通知如下：
    一、会议主题
    贯彻落实全国科技创新大会精神，提升区域原始创新能力，迎接党的十八大胜利召开。
    二、会议内容
    1. 贯彻落实全国科技创新大会精神，推动区域创新体系建设；
    2. 交流地方基础研究工作取得的成绩和经验；
    3. 研究促进地方基础研究工作发展的新举措；
    4. 研究地方开展科教结合促进协同创新的机制。
    三、讨论议题
    1. 如何贯彻落实全国科技创新大会精神，加快基础研究管理体制和机制改革，促进我国基础研究健康持续发展；
    2. 讨论陈部长讲话，研究促进地方基础研究工作发展的新举措；
    3. 探讨如何发挥地方基础研究的优势和特色，交流地方在实验室建设、973计划管理等方面的经验和做法；
    4. 交流地方如何通过科教结合促进协同创新的机制。
    四、参会人员
    各省、自治区、直辖市、计划单列市科技厅（委、局），新疆生产建设兵团科技局主管基础研究的领导1名及相关职能处室的负责人1名。
    五、相关要求
    1. 请各单位按照会议内容和讨论议题开展前期调研，重点做好对深化基础研究管理体制改革，促进地方基础研究持续发展，加快区域创新体系建设的思路调研，认真做好会议交流准备。请将调研材料（见附件1）于10月18日前以光盘形式报送至科技部基础研究司综合计划处。
    2. 为了进一步调动大家的积极性，本次会议拟表彰一批先进工作者和优秀省部共建国家重点实验室培育基地。此表彰不设物质性奖励，不与待遇挂钩。请各单位分别推荐2名从事基础研究管理工作的先进工作者和1个优秀省部共建国家重点实验室培育基地，于10月12日前将相关推荐表（见附件2和附件3）以光盘形式报送至科技部基础研究司综合计划处。推荐基本原则为：
    优秀省部共建国家重点实验室培育基地推荐原则：有规范的管理和运行机制；取得突出的研究成果；对地方基础研究发展发挥了重要带动作用。
    地方基础研究管理先进工作者推荐原则：长期从事基础研究管理工作，工作业绩突出，具有表率作用；对基础研究发展规律有深刻认识，能准确把握基础研究发展形势；在组织973计划等重大项目和国家重点实验室管理等方面具有丰富的经验。
    3. 请各单位于10月15日前将参会人员回执发至jcc@gxsti.net或传真至0771-2631652。
